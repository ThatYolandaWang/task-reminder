# task reminder
根据**高效人士的七个习惯**打造的一款桌面任务应用，帮助用户管理每日最重要的任务，定期提醒，避免被繁杂的任务淹没。

支持windows和macos系统。

前端：react+vite+tailwindcss
后端：rust+tauri

## 本地部署说明
1. 安装nodejs

2. 安装rust环境

    window环境:
    - visual studio developer command prompt
    - https://rustup.rs/
    - 安装rust环境

    mac环境:
    - 安装rust
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
    - 安装tauri
    ```bash
    cargo install tauri-cli
    ```

3. 安装tauri
